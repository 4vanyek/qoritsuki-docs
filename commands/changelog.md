# /changelog (Changelog)

### Аргументы:

#### version

Здесь можно выбрать changelog конкретной версии. Без этого аргумента выводится changelog для последней версии.

**latest**

Выводит changelog для последней версии

### Пример использования:

`/changelog version: 1.1.0`
